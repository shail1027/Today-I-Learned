## C++ `std::unique` 함수


### 개요

C++의 `<algorithm>` 헤더에 포함된 `std::unique` 함수는 컨테이너(주로 `std::vector`) 내에서 **연속된 중복 원소를 제거**할 때 사용한다. 배열의 중복 값을 없애고 유일한 값들만 남겨야 할 때 자주 활용되는 함수이다.

---

### 주요 기능과 반환값

`std::unique`는 이름과 달리 컨테이너에서 원소를 아예 삭제하여 크기를 줄이는 것이 아니라, 중복되지 않은 원소들을 컨테이너의 앞쪽으로 모아 덮어씌우는 방식으로 동작한다.

* **반환값:** 중복을 덮어씌우고 난 뒤, 유일한 원소들로 채워진 구간의 **바로 다음 위치(새로운 논리적 끝)를 가리키는 반복자(Iterator)** 를 반환한다.



---

### 동작 원리 

`std::unique`를 사용할 때 실수하는 두 가지 특징이 있다.

1.  **연속된 중복만 제거한다:** `unique`는 현재 원소와 바로 다음 원소를 비교하여 같으면 중복으로 처리한다. 따라서 떨어져 있는 중복 값은 제거하지 못한다. 배열 전체의 중복을 완벽히 제거하려면 **반드시 먼저 `std::sort()`를 이용해 정렬**을 수행해야 한다.
2.  **컨테이너의 실제 크기(size)는 변하지 않는다:** 앞쪽으로 유일한 원소들을 당겨서 배치할 뿐, 배열의 원래 크기는 그대로 유지된다. 새로운 논리적 끝 이후에 남아있는 원소들의 값은 쓰레기 값(지정되지 않은 상태)이 되므로 접근해서는 안 된다.

---

### `erase-unique` 관용구 (Idiom)

위의 주의사항 때문에, 실제 코딩 테스트에서 중복을 완벽하게 지우고 컨테이너의 크기까지 알맞게 줄이려면 `sort`, `unique`, `erase` 세 가지를 묶어서 사용하는 **erase-unique 관용구**를 사용해야 한다.

`std::vector`의 `erase(시작 반복자, 끝 반복자)` 함수를 사용하여, `unique`가 반환한 쓰레기 값의 시작점부터 벡터의 진짜 끝까지를 물리적으로 잘라내는 방식이다.

```cpp
#include <iostream>
#include <vector>
#include <algorithm>

int main() {
    std::vector<int> v = {1, 3, 2, 3, 1, 2, 4};

    // 1. 먼저 오름차순 정렬을 한다. (연속된 위치로 중복 값을 모음)
    std::sort(v.begin(), v.end()); 
    // v 상태: {1, 1, 2, 2, 3, 3, 4}

    // 2. unique와 erase를 결합하여 중복 제거 및 크기 축소
    // unique가 쓰레기 값 구간의 시작 반복자를 반환하면, 거기서부터 v.end()까지 삭제한다.
    v.erase(std::unique(v.begin(), v.end()), v.end());
    // v 상태: {1, 2, 3, 4} (크기도 4로 줄어듦)

    for (int num : v) {
        std::cout << num << " ";
    }
    // 출력: 1 2 3 4
    
    return 0;
}
```
